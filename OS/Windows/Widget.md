# ウィジェット

Windows11の左下から出てくる場所にウィジェットが表示されます。

## MicrosoftStartの無効化

MicrosoftStartとはウィジェットの右側に表示されるニュースなどです。これらはメディアをブロックしても同じメディアを表示したり、不要なカードを消せないなどのはっきり言ってカスタマイズができないカス仕様となっています。
またこれらのニュースで勝手に通信もするので止めたい場合もあるでしょう。その場合は以下のような手順で無効化可能です。

* ウィジェットを開く
* 右上の歯車をクリックして設定を開く
* フィードの表示/非表示を開く
* Microsoft Startをオフにする

こちらを無効化しても別ウィジェットである天気などは表示可能です。
