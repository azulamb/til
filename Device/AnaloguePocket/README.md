# AnaloguePocket

ゲームボーイの形をしたGB/GBC/GBAが動くゲーム機です。自分でゲームの開発も可能です。

* 公式サイト
  * https://www.analogue.co/pocket
* ドキュメント
  * https://assets.analogue.co/pdf/66b5bf11907751940e6f0d36d5fab2c9/Analogue+Pocket+User+Manual+v1.0.pdf
* ファームウェア
  * https://www.analogue.co/support/pocket/firmware/1.1

## Dock

最新の本体のファームウェアをインストールするとDock接続時にアップデートできるようになったようです。

### ポート

ポートは以下のようになっています。

```
+---------------------------------------------------------------+
| [USB 2]   (Pairing button)   (USB C:Power)   [HDMI]   [USB 1] |
+---------------------------------------------------------------+
```

* USB C:Power
  * 電源
* Pairing button
  * 多分Bluetoothコントローラーのペアリング用
* HDMI
  * HDMI出力ポート
* USB 1
  * 有線コントローラー接続（メイン）
  * PS5コントローラーのDualSenseとかが接続できる
* USB 2
  * 不明
