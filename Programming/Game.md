## 乱数からの方向

rが0-4(上右下左)で来たとき、xとyにそれぞれ+1か-1をする。具体的には以下のような対応関係。

| r | x | y |意味|
|---|---|---|----|
| 0 | x |y-1| 上 |
| 1 |x+1| y | 右 |
| 2 | x |y+1| 下 |
| 3 |x-1| y | 左 |

```
x + ( 2 - r ) * ( r % 2 )
y + ( r - 1 ) * ( ( r + 1 ) % 2 ) )
```
